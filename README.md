```
Usage: termdown.py [OPTIONS] START

  Starts a countdown to or from START. Example values for START: 10, '1h 5m
  30s', '12:00', '2020-01-01', '2020-01-01 14:00'.

Options:
  -b, --blink      Flash terminal after countdown
  -f, --font TEXT  Choose from http://www.figlet.org/examples.html
```
